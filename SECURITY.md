# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**DO NOT REPORT PUBLICLY.**

If you discover a security vulnerability within this project, please send an e-mail to our team at `security@vf9club.vn` (or open a generic issue asking for a private channel).

All security vulnerability reports will be promptly resolved.

### Credentials & Privacy

- **Never commit your real VIN, username, or password.**
- The `.env` file should be added to `.gitignore`.
- This project is a client-side dashboard that communicates with VinFast servers. Usage of your credentials is at your own risk.
